# このリポジトリについて

過去に使用した実績のあるツールやパッケージについて、動作環境を Docker で構築する手順をまとめたリポジトリです。

現状では以下の項目を扱っています。

-|-|-
.NET Core | `3.1` | `/dotnet`
Nginx | `1.17.10` | `/nginx`
Hexo | `1.17.10` | `/nodejs/hexo`
Apache Airflow | `1.10.10` | `/python/airflow`
Diagrams | `1.10.10` | `/python/diagrams`
Flask | `1.10.10` | `/python/flask`
Rust | `1.45.0` | `/rust`
